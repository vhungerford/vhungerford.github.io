# Welcome to Jekyll!

**Hello world**, this is my first *Jekyll* blog post.

I have just started using Github pages

## Key Features

*   **Simple Blogging**: Posts live in the `_posts` folder.
*   **Easy Formatting**: Use standard [Markdown Guide](https://www.markdownguide.org) syntax.
*   **Customization**: Use [Front Matter](https://jekyllrb.com/docs/front-matter/) for metadata like layout, title, categories, and tags.
*   **Code Blocks**: Easily include code with syntax highlighting using fenced code blocks (three backticks):

```javascript
var foo = function(x) {
  return(x + 5);
}
foo(3);
```
